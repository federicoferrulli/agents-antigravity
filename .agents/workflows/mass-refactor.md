# Massive Refactor Workflow
Attiva la modalità Esecuzione Massiva. Ti indicherò una root folder (es. `agents/`) e uno standard da applicare (es. 'Standardizza il tono di voce e formatta i bullet point'). Tu dovrai leggere il contenuto di tutti i file nella directory in autonomia, analizzare le deviazioni dalla regola, eseguire le modifiche in background blocco per blocco e restituirmi solo l'output finale diff per ciascun file modificato. Procedi senza chiedermi conferma tra un file e l'altro.
