# Come Usarlo
Puoi lanciare il comando da qualsiasi directory del tuo PC:

```bash
# Per vedere l'help e le opzioni
antigravity-sync --help
# Per una prova senza modifiche (Dry Run)
antigravity-sync --dry-run
# Per forzare la sincronizzazione sovrascrivendo i file esistenti
antigravity-sync --force
# Per sincronizzare in una directory specifica
antigravity-sync --target ./mio-progetto
```
